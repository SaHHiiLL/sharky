# sharky
make and store ssh sessions. 



## Vision 
 ```$ sharky```

```
1: Linode
2. GCP
3. AWS
....
``` 
```sharky> list```

it should go ahead and grab the first item, and strat an ssh session.


# Valid Commands
```
list
-join <NUMBER OF INDEX>
-create --username <user name> --ip <IP address> --password <password>
```
